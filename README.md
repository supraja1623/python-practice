# python-practice

This repository contains my Python practice files covering all major topics. Each file focuses on a specific concept or area of Python programming, helping me build a strong foundation and improve my coding skills.

## Topics Covered

- `basics.py` – Fundamental syntax and operations
- `list.py` – Working with lists and list methods
- `tuple.py` – Tuple operations and immutability
- `set.py` – Set theory and unique elements
- `dict.py` – Dictionary usage and key-value pairs
- `loops.py` – For and while loops with examples
- `functions.py` – Defining and using functions
- `oops.py` – Object-oriented programming concepts
- `dsa.py` – Data structures and algorithms practice
- `file handling.py` – Reading and writing files
- `testing.py` – Basic unit testing examples
- `mini project.py` – Small project to apply concepts
- `password generator.py` – Random password generator
- `rockpaperscissorsgame.py` – Simple game implementation
- `sample.py` – Miscellaneous practice snippets

## Purpose

This repo helps me track my progress and share my learning journey. It’s also a resource for others who want to explore Python fundamentals through practical examples.

Feel free to explore, fork, or contribute!
